# Iceburg CRM

**Iceburg CRM** یک سیستم مدیریت ارتباط با مشتری (CRM) متن‌باز مبتنی بر
**Laravel** است که به‌صورت **Metadata Driven** طراحی شده و به شما امکان
می‌دهد بدون نیاز به تغییر در پایگاه داده یا کدنویسی سنگین، ماژول‌ها،
فیلدها، روابط و رابط‌های کاربری را **به‌صورت پویا** بسازید و مدیریت کنید.

این سیستم علاوه بر قابلیت توسعه‌پذیری بالا، دارای **قابلیت‌های اختیاری هوش
مصنوعی** است که می‌تواند بر اساس توضیحات شما، **یک CRM کامل را ایجاد،
پیکربندی و حتی برای آن لوگوی اختصاصی تولید کند.**

------------------------------------------------------------------------

## امکانات کلیدی

-   ایجاد خودکار ماژول‌ها، فیلدها، روابط و Subpanelها بر اساس متادیتا
-   پشتیبانی از **Import / Export** در فرمت‌های XLSX، CSV، TSV، ODS، XLS
    و HTML
-   پشتیبانی از **25 نوع فیلد مختلف** همراه با اعتبارسنجی Laravel و
    Masking برای ورودی‌ها
-   دارای **26 تم متنوع** (حالت‌های روشن و تاریک موجود است)
-   **سیستم نقش‌ها و سطح دسترسی** بر اساس ماژول (خواندن، نوشتن، وارد
    کردن، صادر کردن)
-   دارای **تقویم داخلی، گزارشات Audit، نمودارهای Vue 3 و سیستم
    Workflow**
-   پشتیبانی از **روابط چندسطحی** (۲، ۳، ۴ و حتی بیشتر بین ماژول‌ها)
-   API داخلی آماده جهت اتصال به سیستم‌ها و اپلیکیشن‌های دیگر
-   دارای **150 Connector** برای اتصال به سرویس‌های خارجی

## نصب سریع

``` bash
composer create-project iceburgcrm/iceburgcrm iceburgcrm

php artisan iceburg:seed
```

### ساخت CRM با هوش مصنوعی:

``` bash
php artisan iceburg:create --type=ai --prompt="Create a stamp collecting crm" logo="yes"
```

### تبدیل یک دیتابیس موجود به CRM:

``` bash
php artisan iceburg:create --type=adminpanel --connection_host=...
```
